1. **[Naming de Parámetros] Error de PascalCase en parámetro**
   - **Archivo**: `app/Portal.Consultoras.WS.BusinessLogic/V1.1/OfertasBL.cs` línea 428
   - **Problema**: Se está utilizando PascalCase (`FactorNormativo`) para el nombre de un parámetro en el método `SetComponentes`. Los parámetros y variables locales deben estar obligatoriamente en camelCase.
   - **Riesgo**: Violación de convención de código C# (Mantenibilidad / Estándar corporativo).
   - **Impacto en score**: -10 puntos
   - **Tiempo est.**: 2 minutos
   - **Categoria afectada**: Mantenibilidad
   - **Código actual**:

     ```csharp
     private void SetComponentes(Oferta entidad, FichaProductoRequest request, List<MSComponenteResponse> msComponentes = null, int FactorNormativo = 0, bool featureFlagPPUM = false)
     ```

   - **Solución sugerida**: Cambiar el parámetro a camelCase (`factorNormativo`) en la definición del método y donde se utilice dentro del mismo.

     ```csharp
     private void SetComponentes(Oferta entidad, FichaProductoRequest request, List<MSComponenteResponse> msComponentes = null, int factorNormativo = 0, bool featureFlagPPUM = false)
     ```

2. **[Naming de Locales] Error de PascalCase en variable local**

## 📋 Resumen del PR
**¿Qué hace este PR?**
Implementa la lógica para obtener y calcular el PPUM (Precio por Unidad de Medida) de los productos en las ofertas y en el carrito sugerido, para el país Chile validándolo mediante sus respectivos Feature Flags de configuración en la tabla logica. Mapea campos nuevos del precio, contenido, y unidad de medida normativa.

**Archivos principales modificados:**
- [MODIFY] `MSFichaProductoResponse.cs` - Adiciona `FactorNormativo`.
- [MODIFY] `Oferta.cs` - Adiciona `FactorNormativo`.
- [MODIFY] `ProductoCarritoSugerido.cs` - Adiciona estructura `PpumData`.
- [MODIFY] `OfertasBL.cs` - Aplica cálculo condicional de PUM para Chile desde msComponents.
- [MODIFY] `PedidoBL.cs` - Adiciona subconsulta REST a `UrlObtenerPPUMCL` para popular PpumData en carrito.
- [MODIFY] `ConsTablaLogica.cs` y `Constantes.cs` - Nuevos endpoints y format strings.

**¿Qué Podria impactar?**
El flujo de renderizado de la Ficha de producto y Carrito sugerido en Portal Consultoras podrían verse impactados si el feature flag redirige datos o si el microservicio PersonalizacionSearch presenta retrasos.
