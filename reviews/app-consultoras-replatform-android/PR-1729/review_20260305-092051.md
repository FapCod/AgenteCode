## 📋 Resumen del PR
**¿Qué hace este PR?**
Implementa el feature flag `isa_avatar_update` para actualizar el avatar del chatbot ISA en la app Android. Cuando el flag está habilitado por país, reemplaza la imagen estática del chatbot por una imagen dinámica (via Coil desde URL remota) con fallback al nuevo avatar `new_avatar.png`. Aplica tanto en la pantalla de inicio (Standard y NewCS) como en el navigation drawer.

**Archivos principales modificados:**
- [MODIFY] `ChatBotComponent.kt` — Lógica condicional ISA/legacy avatar
- [NEW] `IsaAvatarTileImage` en `NavigationDrawerSubMenuGrid.kt` — Avatar en el drawer
- [MODIFY] `HomeNewCsViewModel.kt` / `HomeViewModel.kt` — Fetch del config ISA
- [MODIFY] `app_general_config.json` — Configuración del feature flag por entorno

**¿Qué Podría impactar?**
- La imagen del chatbot en home y en el navigation drawer cambiará para PE (QAS) cuando el flag esté activo.
- El comportamiento es retrocompatible: sin el flag, UI idéntica a la anterior.

**Impacto:**
- UI/UX, Performance (mínimo — carga lazy con Coil)
