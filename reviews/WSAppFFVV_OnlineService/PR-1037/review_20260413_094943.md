## 🟡 Problemas Importantes (P1 - High Priority)

1. **[Falta validacion de null en retorno de AppSettingsOrchestrator]**
   - **Archivo**: `SomosBelcorpService.cs` linea 32
   - **Problema**: El metodo `AppSettingsOrchestrator.GetAppSetting("SecretKey")` podria retornar `null` si la key no existe en AWS Secrets Manager o en la configuracion local, causando `NullReferenceException` al momento de usarlo para firmar tokens.
   - **Impacto**: Potencial crash en produccion si la configuracion no esta correctamente establecida.
   - **Impacto en score**: 4 puntos (Robustez)
   - **Tiempo est.**: 5 minutos
   - **Categoria afectada**: Robustez
   - **Codigo actual**:

     ```csharp
     private string signingKey = AppSettingsOrchestrator.GetAppSetting("SecretKey");
     ```

   - **Solucion sugerida**: Agregar validacion defensiva o usar el operador de coalescencia nula con un throw explicativo:

     ```csharp
     private string signingKey = AppSettingsOrchestrator.GetAppSetting("SecretKey") 
         ?? throw new InvalidOperationException("SecretKey no configurada en AppSettings o Secrets Manager");
     ```

2. **[Falta validacion de null en credenciales APISB2]**
   - **Archivo**: `RestClient.cs` linea 177-178
   - **Problema**: Los valores de `APISB2_USR` y `APISB2_PWD` podrian ser `null` si no estan configurados, lo cual causaria fallos al intentar autenticarse.
   - **Impacto**: Potencial fallo de autenticacion silencioso o excepcion no controlada.
   - **Impacto en score**: 4 puntos (Robustez)
   - **Tiempo est.**: 5 minutos
   - **Categoria afectada**: Robustez
   - **Codigo actual**:

     ```csharp
     string userApiSb2 = AppSettingsOrchestrator.GetAppSetting("APISB2_USR");
     string passwordApiSb2 = AppSettingsOrchestrator.GetAppSetting("APISB2_PWD");
     ```

   - **Solucion sugerida**: Agregar validacion defensiva:

     ```csharp
     string userApiSb2 = AppSettingsOrchestrator.GetAppSetting("APISB2_USR") 
         ?? throw new InvalidOperationException("APISB2_USR no configurada");
     string passwordApiSb2 = AppSettingsOrchestrator.GetAppSetting("APISB2_PWD") 
         ?? throw new InvalidOperationException("APISB2_PWD no configurada");
     ```

## 📋 Resumen del PR

**Que hace este PR?**
Migra la obtencion de credenciales sensibles (`SecretKey`, `APISB2_USR`, `APISB2_PWD`) desde `ConfigurationManager.AppSettings` hacia `AppSettingsOrchestrator.GetAppSetting()`, que abstrae la fuente de configuracion permitiendo obtener valores desde AWS Secrets Manager.

- [MODIFY] `packages.config` - Agrega paquetes NuGet de AWS
- [MODIFY] `app.config` y `Web.config` - Reformateo XML
- [NEW] Varios `.dll` de CommonAssemblies actualizados

**Archivos ignorados (CI/CD y binarios):**
