**P2 - Medium Priority:**

1. 💡 **`SET_STRATEGY_CODES` definida dentro del arrow function — se recrea en cada iteración**
   - **Archivo**: `src/graphql/server/dataSources/CartAPI.js`
   - **Problema**: La constante `SET_STRATEGY_CODES = [3, 7]` está declarada dentro del cuerpo de `shouldIncludeProduct`, que se invoca en **cada iteración del loop** de productos. Esto recrea el array en heap en cada llamada. Aunque V8 puede optimizarlo parcialmente, la práctica correcta es elevar constantes estáticas al scope superior del método.
   - **Impacto en score**: -1 punto (Mantenibilidad / Escalabilidad)
   - **Tiempo est.**: 2 minutos
   - **Categoria afectada**: Mantenibilidad
   - **Código actual**:

     ```javascript
     const shouldIncludeProduct = (product, item) => {
       if (isFeatureEnabled) {
         const strategyCode = product?.strategySummary?.code;
         const SET_STRATEGY_CODES = [3, 7];  // ← Se recrea en cada llamada
         const isSetStrategy = SET_STRATEGY_CODES.includes(strategyCode);
         ...
       }
     };
     ```

   - **Solución sugerida**: Elevar `SET_STRATEGY_CODES` al scope del método o como constante de módulo.

     ```javascript
     const SET_STRATEGY_CODES = [3, 7];  // ← Definida una sola vez, fuera del arrow

     const shouldIncludeProduct = (product, item) => {
       if (isFeatureEnabled) {
         const strategyCode = product?.strategySummary?.code;
         const isSetStrategy = SET_STRATEGY_CODES.includes(strategyCode);
         const isSetParent = item?.seal?.toLowerCase() === 'set';
         if (isSetStrategy && !isSetParent) {
           return false;
         }
       }
       return true;
     };
     ```
